Copula Default Graph

A bank credit-risk platform that turns transactions, geography, and money-flow communities into correlated joint-default risk — then computes risk metrics at borrower, cluster, and portfolio level. Built to scale to 10M+ borrowers and to be owned by separate teams.

Data Engineer ML Engineer Data Scientist Risk Analyst Platform / Agents

1 · The problem

Each borrower has an individual probability of default (PD). Treating defaults as independent badly understates portfolio risk: borrowers share a local economy (geography), move money between each other (transaction graph), and whole clusters collapse together when a key person — an anchor — defaults. This platform models that correlation explicitly and measures its cost.

4 · The six design decisions that matter

D1 · No dense n×n

A dense matrix at 10M is ~0.8 PB. Dependence is expressed as O(n·K) factor loadings; the loss matrix is computed block-on-demand. Proven: 2M fit in 0.30 s / 96 MB.

D2 · Clusters = segments

Geo/transfer clusters are just persons columns; every metric rolls up by an arbitrary column. New clustering ⇒ no new metric code.

D3 · INV-6 additivity

Segment variance = block-sum of the loss-covariance matrix, never an average of per-borrower ratios (wrong under correlation). Block == dense to 1e-9.

D4 · One metric interface

Dense and factor copulas both expose joint_default_probability_block; the metric layer duck-types on it, so either plugs in unchanged.

D5 · Pluggable PD

Banks usually have a PD model. The platform's value is the dependency layer; it consumes model_pd regardless of origin.

D6 · Anchors add structure

The copula captures co-movement; the anchor detector adds the structural "breadwinner defaults ⇒ family follows" pattern as a conditional-loss uplift.

5 · The core mathematics

Full derivations in METHODOLOGY.md. The essentials:

Expected loss (Basel decomposition)

EL_i = PD_i · EAD_i · LGD_i     Portfolio EL = Σ_i EL_i

Joint default via the latent-variable (Gaussian) copula

D_i = 1 ⇔ A_i ≤ Φ⁻¹(PD_i)     P(D_i ∩ D_j) = Φ₂( Φ⁻¹(PD_i), Φ⁻¹(PD_j) ; ρ )

Vasicek factor model (scales to 10M)

A_i = √ρ_i · Y_f(i) + √(1−ρ_i) · ε_i     corr(A_i,A_j) = √(ρ_i·ρ_j) if same factor, else 0

Multi-factor (geo ⟂ transfer, equally weighted)

A_i = Σ_k β_{i,k}·Y_{k,f_k(i)} + √(1 − Σ_k β²)·ε_i     corr(A_i,A_j) = Σ_k β_{i,k}·β_{j,k} · 1[same factor k]

Loss covariance (the core risk object)

LossCov[i,j] = (EAD·LGD)_i · [P(D_i∩D_j) − PD_i·PD_j] · (EAD·LGD)_j     Var(Loss_S) = Σ_{i,j∈S} LossCov[i,j]

Anchor-contagion uplift (якорный человек)

PD_j | anchor defaults = P(D_j ∩ D_anchor) / PD_anchor     uplift = EL_cluster|anchor-default / EL_cluster

6 · The seven risk-adjusted metrics

MetricFormulaReads as
coefficient_of_variationσ_L0 / ELriskiness, correlation-blind
coefficient_of_variation_copulaσ_L1 / ELriskiness, copula-aware
rarocE[Profit] / Capitalreturn on capital (correlation-blind)
sharpe_indep(EP − rf·Rev) / σ_L0profit vs risk-free opportunity cost
sortino_indep(EP − h·Cap) / σ_L0profit vs hurdle, indep downside
sortino_copula primary(EP − h·Cap) / σ_L1profit vs hurdle, copula downside
sortino_simulated(EP − h·Cap) / σ_L2downside from full simulated tail

The gap between σ_L0 (independence) and σ_L1 (copula) is the correlation penalty. A name cheap on RAROC but expensive on sortino_copula carries hidden contagion risk — the early-warning signal.
